主控板与串口屏之间通过自定义json报文交互。

1. 主控板 -> 串口屏
- Wi-Fi扫描结果

```json
    {
      "json_type": "wifi_ap_list",
      "ap_list": [
        { "ssid": "Office_WiFi", "security": "WPA2-PSK", "rssi": -45 },
        { "ssid": "Guest",       "security": "OPEN",     "rssi": -72 }
      ]
    }
```
        "ap_list"是一个数组，大小不确定，其中包含所有的扫描结果。"rssi"的值为整型。

- 设备地址和版本信息
```json
    { "json_type": "info", "mac_address": "94:A4:08:89:5A:54", "version": "v1  v1.000000.016.238b021" }
```

- modbus寄存器：
    { "json_type": "register", "4" : "0", "5": "90", ...... }
发送的寄存器：4~7, 10~12, 14, 15, 105~110

- 恢复出厂设置消息：
    { "json_type": "cmd", "command" : "factory reset" }
注：这条消息仅在新的LCD硬件（BW16管理屏幕和TOF）上会出现。老硬件上不会有这条消息。

2. 串口屏 -> 主控板：
- 连接Wi-Fi热点
```json
    { "json_type": "conn_wifi_ap", "ssid": "test_ssid", "wifi_pwd": "12345678" }
```

- 断开当前Wi-Fi连接
```json
    { "json_type": "disconn_wifi" }
```


- 测距数据：
```json
    {"json_type":"data","tof_distance":"817"}
```

- 按键事件：
    {"json_type":"data","key_name":"add","key_val":"pressed"}
    {"json_type":"data","key_name":"add","key_val":"released"}
    {"json_type":"data","key_name":"sub","key_val":"pressed"}
    {"json_type":"data","key_name":"sub","key_val":"released"}
